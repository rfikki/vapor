### Requirements:
* node
* gulp (`npm install -g gulp`)

### Setup:
```bash
git clone git@github.com:ethereum/vapor.git
cd vapor
npm install
```

### Development:
```bash
gulp
```
Then open http://localhost:8080

### Building:
```bash
gulp build
```
Builds are found in `dist/`
